# cs50-final (Wayfarer)

Wayfarer is a text-based fantasy adventure game set in the land of Kaelos. It's called _Wayfarer_ because I can't think of a better name right now.

Currently, it has character creation, login, a bit of world to explore, senses, and items. When you create a character, it leads to a town, with a few rooms you can walk around in. And somewhere, there's a cave (that will someday host goblins, a cave troll, and the like.) More coming soon...
