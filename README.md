# cs50-final (Wayfarer)

Wayfarer is a text-based fantasy adventure game set in the land of Kaelos. It's called _Wayfarer_ because I can't think of a better name right now.

Currently, it has character creation, login, a bit of world to explore, and senses. When you login, it leads to an extremely basic town, with a few rooms you can walk around in. More coming soon...

*Note*: The database is NOT uploaded to github, since it contains usernames, etc. A sanitised version will be uploaded later (i.e. without user information.)
